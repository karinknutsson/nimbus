# Nimbus

Nimbus is a combined map and weather app built with Quasar, Mapbox, and the OpenWeather API. It provides an interactive map with live weather data and a responsive interface.

## Features

- Interactive Mapbox map with current weather overlays
- Real-time weather information from OpenWeather API
- Responsive, lightweight Quasar frontend

## Tech Stack

- Frontend: Quasar (Vue.js)
- Maps: Mapbox GL JS
- Weather Data: OpenWeather API

## How to run locally

1. Clone the repo:

```bash
git clone https://github.com/karinknutsson/Nimbus.git
cd Nimbus
```

2. Install dependencies:

```bash
npm install
```

3. Run the app:

```bash
quasar dev
```
